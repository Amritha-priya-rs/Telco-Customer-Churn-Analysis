# Telco-Customer-Churn-Analysis
This project was completed as part of Day 4 of the Data Analytics Bootcamp conducted by TechySpot.

The goal of this project is to build a machine learning model that predicts whether a telecom customer is likely to churn (leave the service) using the K-Nearest Neighbors (KNN) algorithm.

# Dataset

Telco Customer Churn Dataset

The dataset contains customer demographic information, account details, service subscriptions, billing information, and churn status.

# Project Description

Built a K-Nearest Neighbors (KNN) model to predict customer churn using the Telco Customer Churn dataset.

Performed data preprocessing, including handling missing values, encoding categorical features, and feature scaling.

Compared three distance metrics: Euclidean, Manhattan, and Minkowski (p=3).

Tested multiple K values to determine the optimal model performance.

Evaluated the model using confusion matrices and accuracy scores.

Visualized the relationship between accuracy and K values using plots.

Tools Used: Python, Pandas, NumPy, Matplotlib, Scikit-learn, Google Colab.
